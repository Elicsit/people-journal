Sync across devices
Connect a free Supabase database so your journal syncs between your iPhone and computer.
1. Go to supabase.com → New project
2. SQL Editor → New query → Run this:
CREATE TABLE people ( id bigint primary key, data jsonb not null, updated_at timestamptz default now() ); ALTER TABLE people ENABLE ROW LEVEL SECURITY; CREATE POLICY "Allow all" ON people FOR ALL USING (true); CREATE TABLE settings ( key text primary key, value jsonb not null, updated_at timestamptz default now() ); ALTER TABLE settings ENABLE ROW LEVEL SECURITY; CREATE POLICY "Allow all" ON settings FOR ALL USING (true);
3. Project Settings → API → copy URL and anon key
